Molecular Dynamics Simulation Analysis

BIO512 Structural Biology - MD Analysis of Protein Complex Stability

Author

Sayyed Jalil Mahdizadeh

Published

November 10, 2025

1 Introduction

In this practical, we will analyze the results of molecular dynamics (MD) simulations to understand the impact of molecular glue on protein complex stability. Two separate MD simulations have been conducted (~30 ns each): one with the molecular glue (rapamycin) and one without it. MD simulations can reveal how the molecular glue affects the dynamic behavior and stability of the FKBP12/FRB protein complex over time.

The key metric we will focus on is the Root Mean Square Deviation (RMSD) of the protein backbone, which indicates conformational changes compared to the initial structure. Lower RMSD values suggest higher structural stability.

Learning Objectives

By the end you should be able to:

  • Analyze MD simulation trajectories using YASARA
  • Calculate and interpret RMSD values over time
  • Compare protein stability with and without molecular glue
  • Generate and interpret time-series plots of structural parameters
  • Draw conclusions about molecular glue effects on protein complex dynamics

2 MD Analysis with Molecular Glue

2.1 Setting up the analysis environment

  1. Set the working directory:
    • Open YASARA
    • Set the working directory to your/path/MD/with_glue
  2. Set the target:
    • Options > Macro & Movies > Set target > 3FAP_complex.sce

2.2 Running the analysis

  1. Execute MD analysis:
    • Options > Macro & Movies > play macro > md_analyze.mcr > OK
Analysis time

This analysis takes some time to complete. The macro will process the entire trajectory and calculate various structural parameters.

2.3 Interpreting the results

  1. Review the HTML report:
    • After analysis is complete, results will be shown in an HTML file
    • The report contains different analysis metrics with explanations
    • Focus on “Solute RMSD from the starting structure” for protein backbone (RMSDBb)
  2. Understanding RMSD:
    • RMSDBb indicates the protein’s conformational deviation compared to the initial structure over time
    • Raw data is available in 3FAP_complex_analysis.tab

3 Data Processing and Visualization

3.1 Extracting data

  1. Download the analysis file:
    • Download 3FAP_complex_analysis.tab to your local computer (using WinSCP or similar)
  2. Open in spreadsheet software:
    • Open the file using MS Excel or similar software
    • Locate the RMSDBb column and time column

3.2 Creating time-series plots

  1. Plot RMSD vs time:
    • Create a scatter plot or line graph with:
      • X-axis: Time (ns)
      • Y-axis: RMSDBb (Å)
    • This shows how structural stability changes over the simulation
  2. Calculate average RMSD:
    • Calculate the average RMSD over the second half of the simulation (t > 15ns)
    • This represents the equilibrated state of the system
    • Record this value for comparison

4 MD Analysis without Molecular Glue

4.1 Repeat analysis for control system

  1. Set new working directory:
    • Change working directory to MD/without_glue
  2. Repeat steps 2-9:
    • Set target to the complex without glue
    • Run md_analyze.mcr
    • Download and process the analysis results
    • Plot RMSD vs time
    • Calculate average RMSD for t > 15ns

5 Comparative Analysis

5.1 Generating comparison plots

  1. Create comparative visualization:
    • Plot both RMSD curves on a single graph:
      • Line 1: With molecular glue (rapamycin)
      • Line 2: Without molecular glue
    • Use different colors/styles to distinguish the lines
    • Include proper labels and legend

5.2 Statistical comparison

  1. Compare average RMSD values:
    • Compare the average RMSD values from step 9 for both systems
    • The lower RMSD indicates higher structural stability
    • Calculate the difference between the two conditions

6 Data Analysis Questions

Answer the following questions based on your analysis:

6.1 Quantitative Analysis

  1. What are the average RMSD values for the second half of each simulation (t > 15ns)?

    • With molecular glue: _____ Å
    • Without molecular glue: _____ Å

  2. What is the difference in average RMSD between the two conditions? What does this tell you about the effect of rapamycin?

  3. Describe the time evolution of RMSD in both simulations. Do you see different patterns of equilibration?

6.2 Structural Interpretation

  1. How does the molecular glue affect the overall stability of the protein complex? Provide quantitative evidence.

  2. What biological significance does this stability difference have for mTOR signaling?

  3. Are there any periods during the simulation where the differences are most pronounced? What might this indicate?

6.3 Methodological Considerations

  1. Why do we focus on the second half of the simulation for calculating averages?

  2. What other parameters from the MD analysis might be relevant for understanding molecular glue effects?

  3. What are the limitations of this analysis approach? How might you improve it?

7 Advanced Analysis (Optional)

For students interested in deeper analysis:

7.1 Additional metrics

  • Hydrogen bond analysis: Count hydrogen bonds between rapamycin and the proteins over time
  • Contact analysis: Monitor specific residue-residue contacts
  • Flexibility analysis: Calculate B-factors or root mean square fluctuation (RMSF)

7.2 Extended comparisons

  • Different time windows: Compare stability at different simulation periods
  • Per-residue analysis: Identify which protein regions are most affected by the molecular glue
Key Takeaways
  • MD simulations reveal dynamic effects of molecular glues on protein complexes
  • RMSD analysis provides quantitative measures of structural stability
  • Rapamycin stabilizes the FKBP12/FRB complex by reducing conformational fluctuations
  • Comparative analysis between conditions is essential for understanding molecular mechanisms
  • Equilibration time must be considered when analyzing MD trajectories
  • Visual and statistical analysis complement each other in MD interpretation

7.3 Expected Outcomes

Based on the biological role of rapamycin, you should observe:

  • Lower RMSD values in the presence of rapamycin
  • More stable complex with molecular glue
  • Reduced conformational fluctuations when rapamycin bridges the proteins

These results support the mechanism by which rapamycin acts as a molecular glue to stabilize protein-protein interactions and modulate cellular signaling pathways.
